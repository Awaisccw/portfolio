# Modern Portfolio Website

This is a premium, highly professional single-page portfolio website designed for a **Full-Stack Developer & Automation Architect**.

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern Aesthetics**: Deep obsidian theme with sharp gold accents (Amber-400).
- **Animations**: Smooth scroll-reveal effects using Intersection Observer API.
- **Glassmorphism**: Elegant blur effects on navigation and cards.
- **Video Placeholders**: 16:9 responsive containers for project demonstrations.
- **Tech Stack**: Built with HTML5, Tailwind CSS (via CDN), and Vanilla JavaScript.

## Sections

1. **Hero**: Catchy headline with "Available for Night Shift" badge.
2. **About**: Professional summary tailored for junior developer positions.
3. **Technical Arsenal**: Skill tags covering Frontend, Backend, and Automation.
4. **Featured Projects**: 
   - Real-Time React Admin Panel.
   - WhatsApp Server & n8n Automation Bot.
5. **Contact**: Modern form with social links.

## Customization

- Replace `[Your Name]` with your actual name.
- Update social links (LinkedIn, GitHub) in the Contact section.
- Insert your video/image URLs in the project placeholders.
